# PORTFOLIO WEB

### Descripción

Portafolio web desarrollado en React, TypeScript y Vite para mostrar proyectos, experiencia y habilidades de forma rápida y accesible.

### Características

- Interfaz responsiva.
- Páginas de proyectos con demos y enlaces a código fuente
- Mensajería para consultas rápidas y/o solicitudes de colaboración
- Animaciones sutiles y rendimiento optimizado
- Accesibilidad y buenas prácticas SEO básicas

### Tecnologías

- React + TypeScript
- Vite (bundler)
- CSS / Tailwinds

### Instalación

1. Clonar el repositorio:
   ```
   git clone https://github.com/walther-fs/portfolio-web.git
   cd portfolio
   ```
2. Instalar dependencias:
   ```
   npm install
   ```
3. Ejecutar en modo desarrollo:
   ```
   npm run dev
   ```

### Scripts comunes

- npm run dev — servidor de desarrollo
- npm run build — compilación para producción
- npm run preview — previsualizar build localmente

### Estructura del proyecto

- public/
- src/
  - assets/
  - components/
  - hooks/
  - App.tsx
- index.html
- vite.config.ts
- package.json
